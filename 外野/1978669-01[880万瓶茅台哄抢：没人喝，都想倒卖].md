史上最大平价飞天茅台放量季，想要抢到一瓶官方指导价1499元的53度飞天茅台，依然不是一件容易的事。天猫、京东、苏宁易购一到平价茅台的抢购时间，基本都是“秒光”，难度系数与春运抢火车票有一拼。想通过其他渠道抢茅台也没那么容易。不久前，遵义茅台机场发布的一篇12月可以购买平价茅台的活动文章瞬间刷屏——为了买茅台，你得先买机票；而茅台国际大酒店应该是全球唯一一家不愁入住率的酒店，这家位于贵州茅台镇的酒店，因为可以购买平价茅台，未来半年在携程上的房间已经全部被订完。可以说，哪里有平价茅台售卖，哪里就有抢茅台的人。他们闻风而动背后，存在着一张巨大的利益链，每个参与茅台流通环节的人，都能从中分得一杯羹。层层加码后，消费者想要喝飞天茅台，付出的代价是每瓶3000元左右。买到平价茅台有多难？为了在电商平台抢平价茅台，钱强的手机里定了很多闹钟，9点半苏宁易购、10点京东、12点天猫、15点小米有品……每次一到抢购的时间，他就提前两分钟打开页面，一遍遍地重复着加入购物车、结算付款的操作，但没有一次成功，系统前一秒还显示拥挤，后一秒就是库存已不足。自从12月9日得知天猫将大量投放平价茅台后，钱强就成为抢茅台大军中的一份子。他相信自己过去能抢到春运火车票，现在也应该能抢到平价飞天茅台，毕竟天猫、京东、苏宁易购的投放量都不低，加起来超150万瓶。但连续抢了六天，钱强的自信心被碾得稀碎。别人告诉他可以去专卖店试试，他把周边所有茅台专门店都跑了一遍，还是空手而归。“虽然我一开门营业就到了，但店员们都说，平价茅台没货。一个店员和我说，整个北京线下专卖店都没有平价茅台。”AI财经社走访北京多家茅台专卖门店，得到的结果与钱强一样，虽然那些门店里摆着整箱整箱的飞天茅台，却没有一家愿意以1499元的价格往外卖。询问茅台一级经销商有没有平价茅台，得到的结论也是一样，没货。一名一线城市的茅台一级经销商坦言：“我们12月的货还没发，具体什么时候发还在等酒厂的通知。”<img src="https://p1-tt-ipv6.byteimg.com/img/pgc-image/80cd82f0d5d94d7ebd60dd75fd2a9ed0~tplv-tt-shrink:640:0.image" id="aimg_m7N51" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)
近期，茅台集团党委书记、董事长高卫东多次表示，要做好“控价稳市”，要拿出有效措施，让消费者能喝上真正的平价茅台。为此，茅台在四季度加大了直销渠道的投放占比，其中飞天茅台共投放4160吨，按照1吨约等于2124瓶计算，约有883万瓶平价茅台投向市场，占其全年计划的八分之一，是其去年全年投放量的1.57倍。然而，像钱强这样铩羽而归的依然不在少数。这是因为市场需求量远超投放量。以电商平台为例，AI财经社发现，苏宁、京东12月累计投放量分别是50万瓶、60万瓶，从10日开始投放，平均每天放量不到3万瓶。然而，仅12月16日一天，苏宁平台上的预约抢购人数就超过65万人，京东平台上也有68万人。这些预约资格都是要付费成为会员才能获得的。就连12月总放货量只有1.5万瓶的网易严选，每天依然有几万人在抢，前提条件是他们已经花129元成为会员。而国美电商11月搞会员无门槛抢茅台时，当天直接导致APP崩掉。在一些商超渠道，则要花费几千元的真金白银才能获得抢购资格。例如，物美需要提前三个月消费，并且累计金额在3000元以上，才可以平价抢购飞天茅台；大润发设立的门槛是消费要满2000元。据了解，电商、商超等直销渠道从茅台酒厂的拿货价普遍在1299元到1399元，比茅台给一级经销商的供货价969元高出不少。虽然留给这些渠道的利润空间只有一两百元，但他们想方设法也要获得茅台的直销资格，因为茅台是不可多得的引流工具。多位零售渠道商称，将抢茅台的规则定得复杂、设置一定的购买门槛，根本目的是为了将炒茅台的人拒绝在门外，让真正想喝茅台的人能以平价买上。但实际情况是，炒茅台的人依然有办法抢，而喝茅台的人却因规则过于复杂而放弃了。“今天你抢到茅台了吗？”这是最近不少消费者见面的问候语。多数人的回答是：“没抢到，只听说过，没见过。”“已经很久没喝茅台酒了。”多位企业中高层告诉AI财经社，最近一段时间都没有喝茅台，一是因为抢不到，也没有精力去抢；二是因为市场上飞天茅台难以判断真假，不敢买，最终不得不放弃喝茅台。你到底在和谁抢茅台？在抢茅台登上微博热搜榜后，有人发出感慨：“我们到底在和谁抢茅台？”有网友发了一个视频回应，视频里有六个手机，每个手机都在自动抢茅台，还都成功了。AI财经社询问多位酒商，最近有没有抢茅台赚钱，得到的答案都是否定。有酒商明确表示，自己没有组织人抢茅台。“光靠人工很难抢的，不仅要拼手速，还要拼运气，大多数人都抢不到。抢到的很多都是会玩手机、会玩电脑的，还有靠外挂的。”电商平台平价茅台秒光背后，已经形成一套完整的产业链，有人专门通过各种技术手段的应用如外挂、脚本、监测等，在第一时间就抢到了飞天茅台，这本质上和春运抢火车票是一样的。这批人被称为“黄牛党”，他们并不是消费茅台的人，只是垂涎其中巨大的利润空间。在抢茅台串联起的整个利益链上，“黄牛党”属于中间层；靠卖各种抢购攻略、抢购软件赚钱的属于最底层；再往上是酒商们，他们砸入真金白银，靠走量赚钱。可以说，每个参与飞天茅台流通环节的人，都能从中分得一杯羹，层层加码，最终全部由消费者买单。飞天茅台从酒厂出来，第一个环节是直销渠道或一级经销商，他们是茅台最终流通到消费者手里的第一道关卡。目前，直销渠道的售价都是1499元，不加上他们为阻挡黄牛而设置门槛的会员收入等，每瓶利润空间是一两百元，具体拿货价要看各家电商、商超的谈判及议价能力。<img src="https://p9-tt-ipv6.byteimg.com/img/pgc-image/236fe719a27c487098a1be072a833173~tplv-tt-shrink:640:0.image" id="aimg_uL8Xh" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)
按照目前天猫、京东、苏宁公布的投放量看，12月光卖茅台本身，就能给三家分别带来至少0.4亿元到0.8亿元、0.6亿元到1.2亿元、0.79亿元到1.58亿元的利润。到了直销渠道后，谁能抢到飞天茅台凭本身各显神通。有人是利用工作的便利，比如说熟悉京东、天猫、苏宁的平台规则，知道具体的放量规模；有人是找外挂；有人则是发动身边所有亲戚朋友，搞人海战术；还有人真是拼手速……这些人中抢完自己喝的，飞天茅台的流通环节就终止了。但是大量的人都选择再继续流通——要么送人，要么高价卖出去。有消费者告诉AI财经社，自己最近抢购了四瓶飞天茅台，全部以2400元一瓶的价格出售，总共赚了3604元。这样的人算是“羊毛党”，他们全凭运气赚钱，抢到了就赚一笔，抢不到也无妨。但更多的是专业“黄牛党”，他们散落在全国各地，有的蹲守在遵义茅台机场，看到有人提着茅台酒出来，立马凑上前问对方卖不卖，然后开始讨价还价，以一个双方都认可的价格成交；有的人守在网络上，发布自己收购茅台的信息，像个猎人一样等着猎物上门；还有的从羊毛党升级为黄牛党，中间倒一手赚钱。在互联网大厂担任管理层的吴尊，加入炒茅台大军时纯属偶然。开始是自己拼手速，后来抢着抢着就上了瘾，发动周边亲朋好友一起抢。“他们不白抢，抢到的话，一瓶我给他们加五六百，以2000元左右价格收购，然后我再以两千三四的价格卖给二批商，每瓶赚个三四百。”最近这段时间，吴尊每天都能收上来五六瓶飞天茅台，一转手一天就能赚小两千。他号称，一个月倒茅台的收入有几万元，比上班还赚钱。放量效果不明：终端还是每瓶三千，大经销商躺赚不过相比散兵游勇，盘踞在这条食物链顶端的还是经销商。与直销渠道相比，一级经销商的利润空间要大得多，他们几乎不会以1499元的价格向外出售飞天茅台，通常选择自己把钱赚了。前述茅台一级经销商直言：“平价茅台没有，你要拿散瓶飞天茅台，最低拿货价是2700元。”据AI财经社了解，茅台给不少一级经销商的配额是5吨（约10万瓶）、10吨（约20万瓶），这类算“大商”。按照2700元的售价计算，每瓶飞天茅台的利润空间有1700元，5吨配额的利润是0.17亿元，10吨就有0.34亿元。成为茅台经销商往往意味着得到一台“印钞机”，这也导致贵州茅台腐败案频出。过去一年多时间，像袁仁国、刘自力、高守洪、李明灿等十多位茅台高管就纷纷落马。如今随着茅台内部整顿力度加大，想要成为茅台经销商变得越来越难。贵州茅台财报数据显示，今年前三季度，茅台经销商总数只有2049家，较去年减少342家。现有经销商中，新加入的只有14家，还都是系列酒的经销商。够不到经销商门槛的酒商们，则趁着平价飞天茅台集中放量时出手。“我最近收了一百多瓶飞天茅台。”酒商高森透露，他每瓶收价2300元左右，出价2650元左右，一瓶能赚300元，一进一出之间几万元的利润就出来了。另一位石家庄酒商称，最近几天倒茅台，每瓶赚100元，总共也赚了几万元。在这些酒商眼中，这还是小钱。高森称，茅台太占用资金，100瓶就要二十多万元本金，他一般都是拿出三五十万元炒茅台，再高会影响公司的现金流。“我觉得茅台有5个点的利润空间就可以玩，一方面能带来利润，另一方面能引流，方便我卖其他白酒。”与高森相比，西安酒商路伟的胆子大得多，“这一轮放量，我收了几百瓶，小赚了几十万元。”电商渠道集中放量后，很多人找上门要把飞天茅台卖给他。他的收货价是2400元左右，再以2900元的价格卖给熟悉的客户，每瓶能赚500元左右。当然，没有一点资金实力以及客户资源，也赚不了这份钱。一位酒商向AI财经社算了一笔帐，假如以2300元的价格收300瓶飞天茅台，需要占用的流动资金是69万元，如果一瓶赚300元，利润就是9万元，但这建立在把货全部出掉的情况下。AI财经社采访的酒商群体中，还有低价走量的。最近每天收五千瓶左右飞天茅台的贺军称，他做茅台主要是走量，12月15日的收货价是2380元，提前支付出去的资金是1190万元，当天这些酒会经过他手里再卖给其他酒商，一瓶赚10元到20元，一天能赚5万到10万元，“我们的收货价和出货价根据市场行情来。而且是收多少就出多少，不压货，不会囤着待价而沽。”虽然近期有几百万瓶的飞天茅台投向市场，但飞天茅台的一批价很稳定：12月9日到12月16日期间，飞天茅台原箱一批价在2830元左右，散瓶在2420元左右。价格最低的是12月12日，散瓶降到2330元，但后来很快又升了回来。当被问及怕不怕飞天茅台砸在手里时，所有酒商态度出奇一致：不会。一名酒商笃定地说，跌了也没事，大不了自己收藏一段时间，肯定会涨回去的。毕竟与飞天茅台的市场价相比，他们手中的茅台还存在一定利润空间。<img src="https://p26-tt.byteimg.com/img/pgc-image/4f8d6bec10b6415c99f4c41b4dd4b94c~tplv-tt-shrink:640:0.image" id="aimg_i47tn" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)
这个时候最纠结地是小酒商和烟酒店老板。“最近我们停收飞天茅台了，因为市场上放的量太大，我怕击鼓传花砸手里。”一名小酒商谨慎地说。不过从终端来看，相对便宜的飞天茅台还是一瓶难求。位于北京朝阳区的一家茅台专卖店，飞天茅台的正常售价仍然稳稳地停在3050元，即便是买六瓶送一瓶红酒的活动价，也要2799元一瓶。店员傲娇地告诉顾客：“马上要过节了，最近好多人买整箱的送礼，没有折扣，3050一瓶。”（应受访对象要求，文中钱强、吴尊、高森、路伟、贺军均为化名）
