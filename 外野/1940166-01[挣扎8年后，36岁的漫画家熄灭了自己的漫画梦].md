####  黑暗动漫

<strong>“但梦终究是一场梦，是时候该醒了。”</strong>
28岁一头扎进漫画行业，为梦想苦苦坚守8年后，ALEx选择熄灭了自己的漫画梦，离开了他热爱多年、奋斗多年，也磕绊了多年的漫画行业。他的笔名是ALExZHAN13，知道他的人习惯简称他ALEx，是长篇漫画《orz奥兹》、《高玩老爸》，短篇漫画《因为遇见你》、《猫的报恩》的作者。<img src="http://i1.go2yd.com/image.php?url=0PYSqTIL0k" id="aimg_wTk8L" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

6月4日， ALEx在微博发布退坑公告，告别了自己的漫画家人生。<img src="http://i1.go2yd.com/image.php?url=0PYSqTVOwc" id="aimg_U8U88" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

这是无数梦想破灭的悲剧中的一个。远方的理想国犹如海市蜃楼，被生活车轮一样滚滚撵过。生活没有那么多《食梦者》。追逐漫画，也许是太艰难的一件事。<img src="http://i1.go2yd.com/image.php?url=0PYSqTe9O3" id="aimg_xCmCE" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)（《食梦者》是一部讲述漫画家逐梦成功的日本动漫作品）

ALEx爱上漫画的契机并不特别，而他真正与漫画结缘的，是2012年在劲漫画的那枚新人奖。这一年，他的长篇作品《猫的报恩》获奖，这部稚嫩的作品虽作画仍有待改进，但评委的一致好评点燃了的漫画之路。这一年ALEx已经28岁，这是他得到的第一个漫画奖项。<img src="http://i1.go2yd.com/image.php?url=0PYSqTas8I" id="aimg_DGUGZ" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

《猫的报恩》获奖不仅肯定了他以故事为主的漫画创作思路，也是对他在漫画上付出的心血的认可。那一年，大龄新人漫画家ALEx仿佛看到了光，成功拥抱了梦想。这个奖项对ALEx来说是终点，也是走入职业漫画家的起点。他一头扎进漫画。2014年，ALEx的长篇漫画《ORZ奥兹》在《漫画会》成功连载，他摘掉了漫画爱好者的标签，成为一名真正的职业漫画家。<img src="http://i1.go2yd.com/image.php?url=0PYSqTVF3K" id="aimg_pIkyk" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

<strong>“画漫画是自由的，风格和题材是多变的，漫画是有无限可能性的”</strong><strong>“我不需要和他们画一样的东西，我坚信自己会找到自己的方向。”</strong>ALEx成为职业漫画家的年纪已不年轻，这份年龄带来的重量让他感到压力，但他坚信漫画的包容让年龄并不会成为枷锁。创作不分年龄。作品顺利连载让ALEx对未来充满信心， ORZ奥兹在ALEx的努力下发行了单行本。<img src="http://i1.go2yd.com/image.php?url=0PYSqTeeJR" id="aimg_Bppb6" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

（图源：ALEx微博）


有梦想的漫画家不会止步在一本连载漫画，ALEx开始寻求突破。2017年，ALEx拿出《因为遇见你》。这虽然是一个普通的“Boy meet Girl”故事，但ALEx把感情渲染的恰到好处。

<img src="http://i1.go2yd.com/image.php?url=0PYSqT1qGa" id="aimg_zy7td" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

（图源：ALEx微博）

《因为遇见你》是一部能让读者流泪的作品。<img src="http://i1.go2yd.com/image.php?url=0PYSqTuQ5G" id="aimg_kl95m" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

作为漫画作品，《因为遇见你》自然是成功的。但作为参赛作品，《因为遇见你》是失败的。它打动了观众却没能打动评委，ALEx没有入选。幸运的是他有一群老读者，落选的失落被更大的鼓励堆满——<strong>“甚至有读者跟我说，这篇漫画拯救了他，这又让我重燃了信心。我觉得是不是应该创作一些自己更有兴趣，而且有意义的的作品，于是开始着手准备 ‘高玩老爸’。”</strong>跌倒了就再站起来。他再次开始创作。这是ALEx与漫画搭伙的第五年，他33岁。<img src="http://i1.go2yd.com/image.php?url=0PYSqTgW0q" id="aimg_wZoHO" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

可这次，时代没有站在他这边。到了2017年，网媒已经全面兴起，极大冲击了纸媒的生存地位。传统漫画行业受到牵连。很多漫画从纸媒阵地转移到网上，ALEx所连载的《漫画会》也迫于无奈选择休刊，漫画会上的连载作品纷纷自谋出路。<img src="http://i1.go2yd.com/image.php?url=0PYSqTHkBm" id="aimg_cGDva" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

对传统漫画家来说，网络漫画的商业模式是完全不同的一条路。几次商谈的过程说不上顺利，而网络平台的编辑对ALEx的建议令他如鲠在喉。因为市场需求，在网络平台创作漫画的自主性受到了影响——漫画家从作者中心变成了读者中心，向编辑、向市场、向用户妥协。<img src="http://i1.go2yd.com/image.php?url=0PYSqTLdb1" id="aimg_NuyGr" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

遏制创作自主性无异于封杀的梦想，他不想向现实妥协。ALEx没有与平台签约，而是选择将漫画发在个人微博上期望通过微博打赏靠个人作品走到盈利。<img src="http://i1.go2yd.com/image.php?url=0PYSqTeag2" id="aimg_JXa2Z" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

可即便一本漫画只需要5元，他的收入依然寥寥无几。<img src="http://i1.go2yd.com/image.php?url=0PYSqTVGuv" id="aimg_Wj0GH" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

路越来越难走，缺乏平台扶持的ALEx举步维艰。<img src="http://i1.go2yd.com/image.php?url=0PYSqTn5wF" id="aimg_LUK8U" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

再大的创作热情面对两位数的留言，也会化为绝望。<strong>“每次更新前我都信心满满，觉得这话肯定很有意思，但每次迎来的都是更大的失落。现实是每发一话，就更绝望一点。”</strong><img src="http://i1.go2yd.com/image.php?url=0PYSqTP20Y" id="aimg_gF7bN" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

微薄的收入或许能勉强支持他自己，但经不起任何风浪。一旦当家中发生变故，的画笔也不在轻盈。真的太难了，ALEx想到了妥协。当他认真的与网络平台洽淡，决定放下自己的骄傲，成为一名为读者工作的漫画家时，他发现已经没办法弯下腰捡起脚边的六便士。<img src="http://i1.go2yd.com/image.php?url=0PYSqT8nTo" id="aimg_B4mMm" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

<img src="http://i1.go2yd.com/image.php?url=0PYSqTCQcS" id="aimg_tNWYN" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

作为漫画家，ALEx无法接受画风的致敬，更无法接受自己的漫画作品因外部收益而被迫做出改变。他还是拒绝了这份工作。ALEx最终败下阵来，和自己8年的漫画家人生选择说再见。这个决定不是一时冲动， ALEx反思过自己作品的劣势，画过短篇漫画来讽刺自己的处境。“付费问题”“彩色插画”各种问题堆积在自由漫画家的身上，为爱发电的现状对很多作者来说成了跨不过的鸿沟。<img src="http://i1.go2yd.com/image.php?url=0PYSqT8jAD" id="aimg_dB886" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

漫画家是在夹缝里挣扎，只为寻求一点光亮的职业。<img src="http://i1.go2yd.com/image.php?url=0PYSqTFggJ" id="aimg_SMnkN" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

不得不向现实妥协。即便如凌宇沫这样已熬出头的漫画家，也一直对自己的漫画作品耿耿于怀。<img src="http://i1.go2yd.com/image.php?url=0PYSqT43m3" id="aimg_B5vrm" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

和自己和解，向现实低头是一件多么无奈的事情。
ALEx这样被迫放弃漫画的漫画家和爱好者不知还有多少。在为生活奔波后，再回头眺望那一抹希望，梦想如此脆弱。<img src="http://i1.go2yd.com/image.php?url=0PYSqTImZt" id="aimg_Y3F1J" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

专业的技法，角色的互动，这都是作者与读者中的双向选择。读者作为旁观者很难双向选择中找出适当缓冲。在创作过程中，作者也需要适当的服务读者，这其中微妙的制衡点则需要作者自己把控。没人希望优秀的漫画销声匿迹，也没人希望自己喜欢的桥段不见踪影，留下陌生的硬核。<img src="http://i1.go2yd.com/image.php?url=0PYSqTVOwc" id="aimg_svhFd" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

但这都与ALEx无关了。<img src="http://i1.go2yd.com/image.php?url=0PYSqT5Dhs" id="aimg_RE73b" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

这篇退坑公告阅读量比ALEx所有的漫画连载阅读还要高，评论区从未这样热闹过。<img src="http://i1.go2yd.com/image.php?url=0PYSqTPt7h" id="aimg_oSsO0" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

在这篇文章下，ALEx慢慢评论着，和过去的自己道别。生活并不成就《食梦者》。<strong>文：情报姬</strong>

我很讨厌当初一代动漫人对动漫的定义是”爱与梦想“。梦想并没有错，但是当初一代人很极端的鼓吹“画漫画是为了梦想，画动画是为了梦想，做漫画可以自己创造一个理想的世界”等等等等，而导致了一个本来应该是商业行为，主旨应该是“让别人看着高兴”的事情变成了“我要宣泄我的想法，我要表达我想要表达的东西，我在努力去表达，我是与众不同的”，作为商业行为来说，这样的想法明显是错误的。因为人总是喜欢向别人灌输自己内心的想法，而讨厌接受别人向自己灌输——同时还要花钱。很多时候很多作者的创作欲其实就是一种倾诉欲，这种欲望本身并没有错，爱画什么是作者的自由，但是因为这样的倾诉欲导致作品不热门，就要把作者塑造的很悲壮，或者作者自己也相信了这种悲壮，我觉得无疑是害人的，你不能一边一个职业漫画作者不应该是这样的，“让别人喜欢”并不比“创造自己心中的世界”更低一等。

我很好奇中国现在的漫画真正靠漫画本身赚钱的有多少

按个人的理解，中国漫画在2000年之后实体杂志比较红火的时候，商业模式好像还挺良性的，因为是真的有读者买单阅读，作者实打实的能拿到钱

等到了逐步网络化的2010年之后，到现在为止这十来年就没有人能探索出靠谱的商业模式。


理论上比较健康的发展方向，漫画应该走网文那种付费阅读的模式，但是一方面网文的付费阅读模式是很不容易培养起来的，漫画可能没这个机会了，且不说最近阅文的事儿证明了付费阅读可能是个存量很低的易饱和市场，资本可能不愿意再往这类领域砸钱了。就单说从付费/阅读快感/创作成本的性价比来说漫画和网文相比也有天热劣势。网文可以一个人一天写一万字，读者花几分钱，看十分钟。漫画三四个人，一天画10页，还是画面表现力很水的那种，读者花几毛钱甚至几块钱，可能就看一分钟。说得不好听点，除了饭圈，谁会这么花钱？

我没看过数据，但是想也知道，国内的彩色连载漫画，真靠订阅收入就能cover创作成本的，凤毛麟角，估计一只手就能数过来。


所以现在国内彩漫的主流存活（存活而已，赚钱想都不要想）方式是吃补贴，腾讯之类的网站和往漫画行业砸钱的风投不能说多有良心，但至少还是有点担当的，感情和战略投资远大于追求回报。但即使如此，绝大部分彩漫也真是让人没法看……


前几年国产漫画通过将点击人气变现，疯狂的卖了一波漫画原创的ip，结果拍出来的全是垃圾，直接把漫画ip口碑做崩，现在几乎到了改编权白给都没人要的地步。点击人气都是假的，付费意愿才是真的。不敢上架，就别卖ip。


镖人这个漫画我也是刚连载就追了，但这个漫画也没开创什么好的商业模式，作品本身当然挺好的，值得一看，但是能成为现象级作品更多的原因是因为被资本看上了，花了很大的资源在背后推。而且现在这漫画肯定也是完全不赚钱的，一方面全免费阅读一方面实体书电子书各种卖不动，我估计卖的钱连他买那些广告位的费用都cover不了。


漫画这东西难在他的创作门槛比文字高，直接就严重缩减了创作的土壤。但是中国漫画的未来应该还是光明的，因为文创能否繁荣，本质上只和经济发展水平以及文化政策有关。为什么ab站发展之初只能从n站搬mad，而现在每天都有海量原创音乐作品，为什么当初美术生大学一毕业就能进大厂，而现在大公司主美再就业却会被低龄触碾压找不到工作？本质上就是最近二十年因为有一大批家庭吃了经济红利，所以子女从小就获得了80年代之前大部分青少年获得不了的艺术创作教育。只要经济不严重滑坡，文化政策不严重收紧，这方面的教育只会越来普及，年轻人的整体创作基础水平就能不断提升，扩大整个创作的基本盘。即便没有好的盈利模式，表达欲创作欲这种东西本来也是不完全依赖经济条件的，自然会有不愁吃穿的人创作出厉害的漫画。

如果要考虑商业模式的话那首先要搞清楚漫画的定位，既——为了什么去做漫画？


二次元基本的盈利版本是角色和造型，最基本的——一个没有漫画没有动画的原创角色，硬吃造型和设定也是能赚钱的，比如各种和画师合作的“原创企划”周边。这种模式一年少则十来万多则数十万上百万的利润。再深入下去——动画漫画本身除了是商品，也是一个流量和渠道，这个流量就是用来宣传角色，让角色具备更大的商业价值，以角色为根本赚取更大的商业价值。剧情可以**，让你觉得爱与友情，正义和热血是绝对真理，在这个前提下再利用越来越特化的角色来收割观众，而最新的模式就是各种手游的氪金，氪金的本质还是在卖角色，你想要得到这个角色就会去拼命的花钱，利用你喜爱这个角色的心理来赚钱。


那么再回到漫画，商业漫画想要盈利，比起好的世界观，故事，剧情等等，首先最重要的反而是应该要有一套有特色，漂亮的，符合大众审美的人设和背景设定，至少要好看到哪怕你不看这个作品，光看造型就会有“我想买”的想法，在这个前提下再去考虑故事，剧情，最后才是世界观。这些都应该是为商业化服务的，而不是反过来。只有这样——漫画是为了丰满角色，角色丰满是为了盈利，盈利再反哺让漫画可以持续下去。这样才是比较良性的发展。
