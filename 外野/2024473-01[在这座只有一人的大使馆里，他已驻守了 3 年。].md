他是祖国的代言，是冲锋在前的战士。
世界华人周刊专栏作者：牧龙闲人
在遥远的太平洋中部，有一个岛国，叫基里巴斯。
有人说它是世界的起点——赤道和日界线（国际日期变更线）于此交汇，全球新的一天最先从这里开始。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569199/641" id="aimg_WdflT" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


也有人说它是世界的尽头——即使离最近的澳洲大陆，也有2500海里，乘飞机要10多个小时才能到达。
岛上没有报纸，没有电视，没有任何现代化的娱乐设施。
岛民们光着脚，吃着生鱼生虾，过着简陋到近乎原始的生活。
但就在这样一个几乎被世界遗忘的角落，竖立着一面鲜艳的五星红旗。
每个早晨，太阳升起的时候，总会有一个瘦削的身影，高唱着中华人民共和国国歌，将五星红旗缓缓升起。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569218/641" id="aimg_tV2I6" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 吴钟华升旗照
他叫吴钟华，中国“最孤独”的外交官。
独自一人在基里巴斯建立大使馆，以鲁滨逊式的荒岛生活，工作并坚守了3年，创造了世界外交史上从未有过的奇迹。
那段坚强而孤苦的岁月，充满了令人动容的传奇色彩。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569187/641" id="aimg_S2Wfi" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


外交，没有硝烟的战场
1939年，抗日战争时期，吴钟华出生于河北。
战火纷飞的年代，父母为他取名“钟华”，与“中华”二字同音，希望他能成为一个报效国家的有志男儿。
19岁时，吴钟华考入我国著名学府——南开大学。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569207/641" id="aimg_Ra53T" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 南开大学
当时正值美苏两大阵营冷战，国际局势变幻莫测，中国面临着严峻的外交考验。
在这个没有硝烟的战场，外交官是冲在前线的先锋，成功的外交能为国家创造更好的外部环境和发展条件。
鉴于此，吴钟华选择了外语专业，立志成为一名优秀的外交官。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569209/641" id="aimg_yCL3C" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


他发奋读书，除了所修专业课程，还熟识各国历史地理、政治文化，终以优异的成绩从南开大学毕业，顺利进入中国外交部。
但不久后，吴钟华便认识到，外交场远比自己想象的更为复杂和残酷。
在这个人杰辈出、智枭遍地的领域，雄厚的知识储备，仅仅是开展业务的基础。
更多的学问，存在于每个不经意的细节间。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569195/641" id="aimg_b88Qf" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 吴钟华
高深的城府，缜密的话术，不卑不亢的言谈举止，宠辱不惊的情绪管控……在国与国的博弈中，更要懂得何时能进、何时要退，什么应该舍弃、什么必须坚守，能以良好的大局观和清晰的判断力，及时作出最为正确的决断。
纵横捭阖，风口浪尖，一言有失，便可能损害国家的威严和利益。
于是，为了练好外交本领，吴钟华兢兢业业，全身心地投入到工作当中。
工作之余，他会有意识地保持严肃状态，每天对着镜子，模拟不同场合下的涉外交流。
严苛的自我磨砺，让他练就了一身强大的气场和严谨的对话意识。
他先后在礼宾司、国际司、对台办等部门任职，因表现突出，1975年开始出国担任驻外使节。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569194/641" id="aimg_H1vvU" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


巴基斯坦、斯里兰卡、马尔代夫，这些地方都曾留下过他的足迹。
在经过20多年的历练之后，吴钟华调任中国常驻联合国代表团，为中国在国际舞台发声，展现大国威仪。
粉碎**势力的阴谋1989年5月，吴钟华受外交部委派，飞赴太平洋岛国斐济，任驻馆大使，同时兼任基里巴斯和瓦努阿图两国大使。
当时，台湾当局以“金钱外交”和“弹性外交”政策，与我们争夺外交阵地，并不断向基里巴斯和瓦努阿图示好，妄图利用这些岛国，获得一些国际上的认同和支持。
为了粉碎台湾当局的阴谋，吴钟华先后8次往返于斐济和基里巴斯、瓦努阿图之间，与**势力斗争。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569196/641" id="aimg_r0869" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 基里巴斯
在多次打掉台湾当局的“建交”图谋后，吴钟华向外交部提议，应尽快建立中国驻基里巴斯大使馆，以稳固这块外交阵地。
提议很快获得了批准。
1990年2月，51岁的吴钟华只身前往基里巴斯首都——塔拉瓦，筹建中国大使馆，从此开始了一人一馆的苦行僧般的生活。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569188/641" id="aimg_MAszz" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


一个人的使馆
基里巴斯是经联合国认定、世界最不发达国家之一。
它由33个大小岛屿组成，陆地总面积只有811平方公里，但地形狭长，东西连绵约3870公里，南北约2050公里。
这令它成为了全球唯一一个同时地跨南北纬和东西经的国家。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569193/641" id="aimg_OIX0s" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


岛上生活艰苦，居民大多住在棕榈叶制成的棚屋内，目之所及，除了椰子树和白沙滩，便是一望无际的太平洋。
因条件所限，吴钟华临时租用的馆舍，只是一栋草房。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569232/641" id="aimg_a149s" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


为了尽快开展工作，他顶着赤道骄阳，马不停蹄地准备开馆事宜。
钉馆牌，竖旗杆，买保险柜，安装通讯设备……
从会场布置到食物供应，从打印请柬到邀请各路要员，一切都是亲力亲为，毫不含糊。
大使馆是一个国家的对外门面，可以简单，但决不能有损国家形象。
终于，在没日没夜地忙碌了三天之后，28日下午6点，大使馆开馆招待会，正式拉开帷幕。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569198/641" id="aimg_XxOA6" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 吴钟华与国内考察组成员，在驻基里巴斯大使馆门前合影
招待会上，基里巴斯总统夫妇及内阁成员悉数参加，对这位到任的中国外交官表示热情欢迎，英国、澳大利亚、新西兰驻基里巴斯高专亦到场庆贺。
在得知招待会的所有工作都是吴钟华独自一人完成的时候，所有人都惊叹不已。
宴会食物以生鱼虾为主，也有一些面食，吴钟华从斐济使馆背来的几听中国罐头，成了餐桌上的“特色大菜”，深受客人们的欢迎。
招待会从下午6点一直开到了晚上11点，在送走最后一位客人后，吴钟华才发现衣衫已被汗水浸透了。
他累得双腿僵直，坐都坐不下。
但他还不能休息，第一时间发出消息，向全世界宣告，中国驻基里巴斯大使馆正式成立！
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569190/641" id="aimg_wMKUm" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


炸弹，车祸，和盗贼
招待会圆满结束后，吴钟华的工作也步入了正轨。
大使馆虽然只有他一个人，但麻雀虽小五脏俱全，国内与基里巴斯的各项事宜，都由他一个人处理。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569203/641" id="aimg_ADQ9J" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


他身兼数职，工作量之大可想而知。
作为馆长，两国间的政治、经济、文化等诸多方面事务，是他的首要工作。
作为馆员，使馆的礼宾交际、拜会宴请、联系接待，他也责无旁贷。另外，他还是会计、旗手、厨师、司机、采购员、招待员……
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569202/641" id="aimg_kjAn5" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 1990年3月，刚建馆时，吴钟华在临时使馆馆牌和带国旗汽车前的留影
比起一般的大使馆，吴钟华还有一项非常繁重的工作，那就是使馆的一切建设，都要从头做起：
他要设计使馆的大门、篱笆、车棚等，满岛去买建筑材料、找人施工，有时候还要亲自干。
虽然累，但他从来没有怨言。
在吴钟华看来，大使馆就如同祖国的领土，这里的一草一木、一花一石，他都要用心呵护。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569208/641" id="aimg_FIM6E" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 吴钟华在使馆栽树
然而危险也常常不期而遇。
一次，当地警察局突然通知，在使馆附近发现了一枚500磅的炸弹，需要他立即撤离！
原来，二战期间，这座小岛曾被日军占领，后又被美军攻入，一周内接连投下了3000吨炸弹。
以至于战后几十年，岛上还不准焚烧垃圾。
吴钟华只好离开使馆，饿着肚子在外流浪了一天，待拆弹专家排除隐患，才得以返回。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569197/641" id="aimg_w90pg" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


还有一次，吴钟华在海边散步，突然感觉被什么东西撞了一下，随后便失去了意识。
醒来后，才知道是一个骑摩托车的酒鬼撞倒了他。
从周围人的口中，吴钟华得知就在两周前，有人在这儿因车祸而死。他这才感到后怕：
“如果我死了，保险柜内的数万澳元建馆费和文件，岂不是都成了死案？”
他赶紧返回大使馆，将保险柜的开法用中文写在了工作笔记上，这样，一旦自己再出事故，国内前来处理后事的人，就能顺利接管自己的工作了。
在他眼中，国家的利益，远远要比自己的生命更重要。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569206/641" id="aimg_vNmFx" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


大使馆也曾遭遇盗贼。
那天，他外出参加招待会，半夜返回，一开院门，便听到屋后有开窗奔跑声。
他大吼一声“谁”，抄起门后的棍子便追了过去，然而夜色漆黑，贼人已然逃得没了踪影。
他匆匆冲进屋子，发现保险柜完好，这才松了口气。
他把所有灯全部打开，犄角旮旯搜寻了一遍，所幸损失不大，只坏了一扇窗子，丢了抽屉里的几十澳元。
那夜，他的灯亮了一宿，一刻也不曾入睡。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569191/641" id="aimg_zoJqY" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)

吃树叶、挖青苔的清苦岁月
基里巴斯所辖多是低平的珊瑚岛，土地贫瘠多沙，每年有10个月以上连续干旱少雨。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569225/641" id="aimg_u8484" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


受自然条件限制，这里无法种植蔬菜，当地人也没有吃蔬菜的习惯。
但对于中国人而言，没有蔬菜简直是噩梦。
每两个月，岛上会有一艘澳大利亚的商船运来食品和日用品，只有这时，吴钟华才有机会储存一些蔬菜和水果。
但吃完之后，又是长达一个多月的漫长等待。
一个偶然的机会，吴钟华听说当地的一种树叶可以食用，于是每次熬汤的时候，便摘几片嫩叶，扔进汤里解馋。
有时也会去挖一些青苔，洗净晾干，偶尔吃上一点儿。
树叶也好，青苔也罢，味道终究是比真正的蔬菜差得远的。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569216/641" id="aimg_cgag8" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


饮食虽能将就，但独在异乡，对亲人的思念和对故土的眷恋，却让吴钟华每每于夜里神伤。
但他没有任何人能够倾诉。
1990年4月23日，吴钟华收到一封家书，得知父亲已于3月30日过世。远隔千山万水，这封信用了足足24天，才送到了他的手上。
他心中万分悲痛，但当天下午，正是基里巴斯法院新办公楼揭幕仪式，他在受邀之列，只得强颜欢笑，出席了这次活动。
晚上回来后，他独坐灯前，捧着这封家书，泪流满面。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569210/641" id="aimg_W7355" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


自古忠孝不能两全，这句话说来轻巧，但它的背后，蕴含着多少英雄儿女的辛酸血泪啊！
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569217/641" id="aimg_zw5wD" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


最后一次升旗
1992年，吴钟华接到调令，踏上了回国之路。
在这座只有一人的大使馆里，他已驻守了3年。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569205/641" id="aimg_C66Zv" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


从仅有草房一栋，到建成最标准的馆区，五星红旗在椰树林里飘扬，步道通向海边，道旁繁花似锦。
他把中国大使馆展现给了基里巴斯，也把中国和中国外交官的优良风貌，展现给了整个世界。
离任当日，吴钟华坚持在使馆门前最后一次升起了国旗。
那次的国歌，他唱得格外悲壮、格外铿锵有力。
这是他一生中最艰辛、最不平凡的3年，以一段中外外交史上从未有过的孤苦岁月，光荣完成了祖国赋予他的使命。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569192/641" id="aimg_yrERC" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 吴钟华在白布上一针一针的缝出“庆祝中国与基里巴斯建交10周年”中英文字样
也在中国外交史上，写下了浓墨重彩的一笔。
离开基里巴斯后，吴钟华依然奋战在外交一线。
他调往驻美国洛杉矶总领事馆工作，任副总领事，后任中国外交史学会会员，以外交官的身份，悉心钻研指导我国的外交历史。
他是一个赤诚而无私的爱国者，对祖国有着无比深沉的热爱。
<img src="http://inews.gtimg.com/newsapp_bt/0/13945569213/641" id="aimg_xCC3w" onclick="zoom(this, this.src, 0, 0, 0)" width="680"/)


● 晚年吴钟华（图片来源：金日说；摄影：杜辉煌）
如今，我们的国家在国际舞台扮演着越来越重要的角色，得到了越来越多的赞誉。
这其中自然离不开吴钟华大使这样无怨无悔、在异国他乡为祖国默默奉献的外交官们。
国际局势风起云涌，在这个没有硝烟的战场，他们是祖国的代言，是冲锋在前的战士，于唇枪舌剑中，捍卫着国家的利益和尊严。
