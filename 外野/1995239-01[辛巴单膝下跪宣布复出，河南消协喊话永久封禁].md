id="aimg_ivyDY" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" src="https://inews.gtimg.com/newsapp_bt/0/13329757392/1000" id="aimg_awPhP" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)此前，快手主播与快手电商业务，一直以来都是利益共同体。快手早期的流量分配机制和平台文化，让快手的电商直播，成了一门“家族”生意。在快手上，有“六大家族”的主播矩阵，拥有超过8亿的快手粉丝，以辛巴为代表的“818家族”影响力最大。2018年起，快手逐渐加速“去家族化”。2018年7月，快手推出MCN合作计划，扶持新主播和商家号，以及引进明星和文化名人。2019年，快手宣布用100亿流量，扶持10万优质生产者，重点覆盖20个垂直领域，将流量分散至中腰部，丰富社区生态。毋庸置疑的是，拥有超7000万粉丝的辛巴，曾给快手带来过极高的流量。据去年的618达人榜数据显示，辛巴家族中的辛巴、初瑞雪、蛋蛋包揽前三，家族成员在前10榜单中占据6席。在这份榜单中，罗永浩排名第9，李佳琦排名第20。但“蜜月期”早已过去。2020年4月，辛巴曾与快手另一头部主播“散打哥”发生矛盾，引发粉丝骂战，最终快手官方下场处理，对战双方也先后宣布暂时退网。在退网初期，辛巴传出将入驻另一短视频平台的消息。与平台叫板让辛巴更加“出圈”，也让他与快手产生了裂痕。辛巴的傲慢不仅如此，怒怼品牌方、单方面要求品牌方送粉丝耳机、抨击合作明星张雨绮、怒斥保安等都登上热搜......这一系列事件，直至糖水燕窝事件，也让平台遭遇了巨大的信任危机和舆论压力。本次辛巴宣布回归直播带货，可他依赖的快手平台，从营收上，已经不太依赖直播业务了。<img src="https://inews.gtimg.com/newsapp_bt/0/13329757513/1000" id="aimg_ubEdG" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)快手营收主要由直播、线上营销服务与其他服务三大组成。2020年全年，直播带来了332亿元营收，仍为公司最大收入源。不过，在总营收占比已从2019年的80.4%下降到56.5%。而线上营收服务，即互联网广告收入，2020年的营收为218.5亿元，同比增长194.6%。在总营收占比已从2019年的19%上升到37.2%。<img src="https://inews.gtimg.com/newsapp_bt/0/13329757514/1000" id="aimg_PLzHO" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)如果仅看2020年第四季业绩，线上营收服务营收已经超过直播业务，在总营收的占比高达47%，高于直播的43.6%。快手在财报中指出，由于广告业务占比提升，也带动了公司毛利率从2019年的36.1%提升至40.5%。<strong>河南消协喊话永久封禁</strong><strong>人民日报转发表态</strong>“糖水燕窝”事件后，辛巴几乎成了直播带货行业乱象的一个“代言人”。但对于复出，他一直没停止过试探。2月25日起，辛巴陆续出现在辛选旗下主播的直播间，还恢复了在各大直播间疯狂刷榜。看到反响还不错，他紧接着就透露了自己将于3月底到4月初复出的计划。但直播电商行业，也不再是辛巴当年纵横时的那片江湖了。一系列“打假”风波过后，野蛮生长的电商直播行业开始朝正规化发展。3月18日，中国广告协会发布了[《网络直播营销选品规范》](http://www.yousuu.com/name/%E7%BD%91%E7%BB%9C%E7%9B%B4%E6%92%AD%E8%90%A5%E9%94%80%E9%80%89%E5%93%81%E8%A7%84%E8%8C%83)，涉及商家、商品资质、质量检验把控，商品直播表述等，规定主播需如实描述商品信息，不得对商品信息进行夸大。今年的“315”晚会，国家市场监督管理总局发布的[《网络交易监督管理办法》](http://www.yousuu.com/name/%E7%BD%91%E7%BB%9C%E4%BA%A4%E6%98%93%E7%9B%91%E7%9D%A3%E7%AE%A1%E7%90%86%E5%8A%9E%E6%B3%95)，对直播带货做出了专门要求，其中规定了直播视频自直播结束之日起至少要保存三年，还规定经营者应当以显著方式展示商品或服务具体信息等诸多措施，这几乎都戳中了大主播们的七寸。当年和辛巴一起同为网红主播“三国杀”的李佳琦和薇娅，也已经走上了不同的商业化道路。而且，在辛巴退网的期间，不仅粉丝掉了200万，其旗下拥有988.8万粉丝的主播安若溪也被曝退出了“818家族”。这也说明，若辛巴回归不能再现往日辉煌，旗下主播收益减少就可能会选择离开。就在辛巴家族的粉丝们热切期盼着辛巴复出的时候，河南消协也在一次媒体通报会中公开喊话，由于辛巴损害了众多的消费者利益，希望永久封停辛巴的快手账号。而这次喊话很快也得到了人民日报的转发。2月26日，河南省消协举行媒体通报会时表示，河南省消协希望直播平台对辛巴团队的处罚不是封停60天账号，应是永久封停，不给那些不良主播提供直播平台。河南消协对于辛巴一事的处理态度很坚决，河南消协的负责人表示，虽然辛巴团队和产品方均在广东省，但是他们认为此举损害了河南消费者的利益，这事消协就要管到底。所以，对于辛巴而言，不管是直播还是带货，似乎都很难再回到“糖水燕窝”事件之前。
