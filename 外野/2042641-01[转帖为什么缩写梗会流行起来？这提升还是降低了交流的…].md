<strong>02 为啥会出现如此逆天、“妈见打”的缩写梗呢？</strong>
反正我和老爸老妈用缩写时，他们都恨不得顺着电线杆子爬过来拿鸡毛掸子抽我一顿：好好中国话不讲，汉字不讲，讲什么火星文？！
缩写梗是一种文化现象，普遍流行于 00 后，因为它主要通过自媒体，特别是短视频媒体传播，它已经成为一种不可忽视的亚文化现象。语言是联通人与人思想的最重要的媒介，语言的重要性再怎么强调都不为过，很多人把缩写梗只是当作一个年轻小众文化的“圈地自萌“而不予理会，把缩写梗视为年轻人“不好好说话”的证据。有的人持悲观想法，认为长久以往，年轻人最后会连汉字都不会使用，把传统汉字的意蕴忘得一干二净。有的人持乐观想法， 认为缩写梗毕竟是网络圈，而且是极小部分的年轻人网络圈的内部加密语言，它的波及范围有限，对现实生活的影响不大。毕竟如果现实中有人对你说：“xswl，psq 的 xjj 发的那条 news 也太阴间了，xbhl”（用拼音念出），你觉得会觉得这个人是智障。而且你敢对你老板说“dbq，老板，我再 run 一遍程序。”？？？！（老板：你哪国家的混血啊？）
难道真的有人会在语文试卷的作文上写：“今天，风和日丽，万里无云，一切都很绝绝子。我们全家去湖边烧烤，烤五花肉，yyds！”……
我们可以看到，缩写梗有它的范围，它范围是限定的。从空间上看，它主要存在于网络空间中，而非现实生活中。但很多缩写其实已经“破壁”了，也就是说，它们已经从网络空间渗透到现实生活。从使用范围上看，它主要停留在口头语言，目的是日常沟通，而非书面语言。这里的书面语言指一种相当正式的文体，不单是指写下来的语言。

<strong>

03 缩写梗可能的哲学问题</strong>
<strong>1- 语言巴别塔理论</strong>
《圣经·创世纪》第 11 章记载，人类想建造一座塔，塔顶通天。
“他们说、来吧、我们要建造一座城、和一座塔、塔顶通天、为要传扬我们的名、免得我们分散在全地上。”
神毁了这座塔，为了阻止人类以后再建这样的塔，神变乱人的语言，使人类互相之间不能沟通，计划因此失败，人类从此散落在全地，说着不同的语言。
因此，巴别塔（希伯来语：מגדל בבל‎）在希伯来语中有“<strong>变乱</strong>”的意思。

<img src="https://pic1.zhimg.com/v2-10093ebeda4897806e4933b753ddf077_720w.jpg?source=8673f162" id="aimg_bw9AN" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)
巴别塔（图来自网络）
巴别塔是一个古人用来解释语言为何产生的神话。<strong>巴别塔象征着人们共享同一种语言的完美状态，抑或是语言的乌托邦。语言巴别塔便象征人们试图建立统一语言的努力。</strong>此后，人类建造语言巴别塔的努力层出不穷。
俄国世界语作家柴门霍夫（Lazaro Ludovik Zamenlof）创造出了一门新的语言——世界语，他称其为 Esperanto，世界语顾名思义，就是出于让全世界的人通用的目的而创立的。当时，世界语很是风靡了一阵。我们熟知的鲁迅、巴金，到爱因斯坦、高尔基，这些中外文学家都会说世界语。
<img src="https://pic3.zhimg.com/v2-206655879ef3565cca5d55b167ca7c51_720w.jpg?source=8673f162" id="aimg_CPXdn" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

当代分析哲学家们提出的<strong>意义逻辑指称论</strong>也可以看作是建造语言巴别塔的努力。
所谓意义逻辑指称论，就是名称和指称对象的严格对应，该名称的意义就在于它和某个外在指称对象的严格相符。比如“苏格拉底”这个专名，它的指称对象就是现实中的苏格拉底。
按照这种理论，语言相当于世界之图画，语言是对世界和实在的描绘，而且是纹丝不差的精准描绘。

<strong>2- 语言图画论和语言游戏</strong>
语言图画论追求的是语言和世界或实在的严格对应，语言的意义就在于这种特定的对应关系，即指称关系中。
<img src="https://pic2.zhimg.com/v2-f9b66679573c6506c6e9786b51e1c6d6_720w.jpg?source=8673f162" id="aimg_XK6If" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)
语言的逻辑指称论图示
维特根斯坦前期相当痴迷于语言图画论，试图追求语言的确定性。但在后期，维特根斯坦意识到了语言图画论的问题。在分析日常语言时，他发现语言图画论不再适用日常的语言使用情况。因为日常语言是动态的、鲜活的，语言和世界的关系也处于动态的变化之中，而非静态的一一对应关系。
<img src="https://pic1.zhimg.com/v2-3ece04c62b8a0aa1035b35ffc38ce28f_720w.jpg?source=8673f162" id="aimg_dtMI5" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)
路德维希‧约瑟夫‧约翰‧维特根斯坦（Ludwig Josef Johann Wittgenstein，1889 年 4 月 26 日—1951 年 4 月 29 日）
维特根斯坦把语言的使用看作是游戏，提出了着名的<strong>“语言游戏”理论</strong>。简单来说，<strong>“语言游戏”认为人们使用语言的活动本身就是游戏。任何游戏都有游戏规则，语言游戏也有其自身的规则，即语言用法。只有把握游戏规则，人们才能进行游戏。语言也是如此，只有掌握语言用法，人们才能学会使用某种语言。</strong>
<img src="https://pic3.zhimg.com/v2-596ecb5efb4186ce728ca8945476e0eb_720w.jpg?source=8673f162" id="aimg_H2ntR" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)
Language-Games
日常语言中人们使用语言的方式不是唯一的，而是多样的，依据不同的情境而变化。维特根斯坦说，我们通常一边玩一边制定规则。只有在语言活动中，语言和世界才真正发生了关系，语言的意义不再是语言对实在的严格指称，而是变成了语言的使用。<strong>语言和世界统一于语言的使用中。</strong>
<img src="https://pic1.zhimg.com/v2-1be1b3ea95649586a328195f5ab8e0f6_720w.jpg?source=8673f162" id="aimg_bWwKu" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

<strong>04 缩写梗：一种新的语言游戏</strong>
根据维特根斯坦的语言游戏理论，我们可以将缩写梗视为 00 后年轻人独创的一种新的语言游戏。这些语言的意义不在于它们的内容，也不在于它们指称的对象，而在于它们<strong>是如何被使用的。也就是说，我们应该关注的是这些语言和世界是如何互动的。</strong>
拿最经典的 yyds（永远的神）来说，它首先具有<strong>私密性</strong>，对于那些不知道 yyds 是什么意思的人来说，ta 可能就无法理解下面这句话：这部动漫 yyds。其次，它具有<strong>含糊性</strong>。yyds 不是一个确定的名称，它不是名词、动词、形容词，以及其它词性，我们说不出它到底是个什么类型的词。但是如果非要归类，我们会把它归为形容词一类，在英语语法中，则是状语。含糊性的好处就是它可以指代大多数事物，可以笼统地表述为：X yyds （X 为任意值）。
缩写梗还有一个很重要的特征，即它的<strong>安全性</strong>。拼音缩写最早开始使用的一个很重要的原因是避开“敏感词”审查。拼音缩写很好地避开了机器审查的火眼金睛，可以说是广大网民和官方不断斗智斗勇的成果。没想到这种“躲避”矫枉过正，拼音缩写一发不可收拾，蔓延到了日常白话语领域，从而形成了一股浩浩荡荡的拼音缩写风潮！现在，不懂个缩写词，都不好意思说自己是 00 后。
缩写梗已经成为一种新的语言游戏。因此，为了学会这门“语言”，我们必须了解它的游戏规则。缩写梗的游戏规则有哪些？通过观察，我们可以总结出以下规律：
<strong>汉字词组拼音首字母缩写（为主）</strong>。比如 ssfd，xbhl， yyds。 Nsdd（你说得对），bhys（不好意思），人名如 xz， zs， wyf， wyb。<strong>英语首字母缩写。</strong>如 gov， gj （good job）， wd（well done） ，nbcs（nobody cares）<strong>英 + 汉首字母缩写组合。</strong>如 rxl（really 心累），rnb（really 牛逼）。<strong>英 + 任意方言（粤语为主）+ 汉字首字母缩写组合。</strong>如 bhs（不嗨森），muji（母鸡啊）<strong>其它汉字。</strong>如 bp（白嫖）、dw（毒唯）、fw（腐唯）、kdl（嗑到了），还有数字缩写 271（爱奇艺）、227（肖战事件相关者），绝绝子，安利，种草，奈斯，火钳刘明，双厨狂喜，瓜保熟吗，影流之主。

这一类富有很强的亚文化圈色彩，而且多是新造词。
这类没有规律可循，只能靠长期网络浸淫，但因为是汉字，连蒙带猜，基本能明白是几个意思。

<strong>05 缩写梗：“语言上的残废”？</strong>
我们先来看大家对缩写梗的种种意见。
<blockquote>“网络流行语只是为了好玩，但它们并没有因为语言的缺乏而改变。”

“当时杨倩获得了冠军，许海峰为她写了 yyds 送给她，yyds 不能表达感情？”

“有些网络词汇很不错，但是英文缩写之类的词汇真的很难懂，你必须时不时地检查一下这个词的意思。”

“我女儿经常用这些网络流行语，我不反对。中国文化博大精深，你为什么不学习唐宋诗呢？用字母代替好的汉字，真是无聊透顶。”</blockquote>

<img src="https://pic2.zhimg.com/v2-95a4ab2495ba6f144d253552f6b75cfc_720w.jpg?source=8673f162" id="aimg_D7GlE" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)

<strong>意见繁多，不一一列举。大致来看，可以分为正反两方。</strong>
反方认为：<strong>缩写梗消解了汉语的严肃性，是语言缺乏的表现，不利于人们学习博大精深的汉语。</strong>其根本忧虑是：缩写梗只会让人变成“语言上的侏儒”或“语言上的残废”。
正方认为：这只是一种网络流行语，每一代都有都有一代的流行语。像 90 后当年也沈迷于非主流火星文无法自拔，没必要对 00 后这么苛刻。其根本立场是：<strong>我们应该对亚文化抱以更多的尊重和理解。</strong>
中立方：I don’t care。
不得不说，三方各有各的理。就如这些缩写梗本身带着防窥视的文化壁垒一样，不同的立场之间也存在着厚实的壁垒。但对于一种已经“出圈”的缩写梗来说，它既然已经成为公共现象，我们还是可以讨论一二。
<strong>维特根斯坦认为语言的意义在于语言和世界的互动。如果把缩写梗看作是一种新的语言游戏，那么它的意义便在于它的使用。缩写梗以拼音缩写，外来字符缩写替代完整的汉字词组，这不仅仅消解了汉字的严肃性，更是消解了汉字本身。汉字是世界上罕见的象形文字，和其它语音文字不同，象形文字首先是表形，其次是表意。意由形生，形由意达。拼音缩写彻底打破了汉字的传情达意之美，而将其消解为字母缩写，把表形文字变成了表音文字，不得不让人深思这或许也是现代性对中国传统文化的冲击的表现。缩写梗用冷冰冰的字母缩写代替完整的词组，侧面也映射了现代人对社交距离的敏感。我们总是有种直觉，“不好意思”更具有人情味，由于汉字的表意特征，透过“不好意思”四个字，我们似乎能透过文字看到说话者的愧疚的动作、情态。而 bhys 则将说话者的情感完全隐藏在意味不明的四个字母中，我们不管怎么凝视 bhys，都无法窥视说话者的内心感受。</strong>
维特根斯坦说，<strong>“语言的极限意味着我世界的极限。”</strong>
